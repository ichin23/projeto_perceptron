Questão 5: Estratégias para Aumento de Performance

  1. Otimização do Espalhamento (sigma): Ajustar a largura de banda gaussiana via busca em grade (grid search) para encontrar a melhor interpolação nos limites dos dados de treino.
  2. Treinamento de Centros para Ambas as Classes: Atribuir centros RBF para cobrir também as regiões da classe negativa (-1.0), evitando zonas de sombra de dados onde a rede infere apenas o bias de forma genérica.
  3. Variação de Topologia (Aumentar M): Incrementar a quantidade de neurônios ocultos M. Ao projetar para dimensões maiores, a separabilidade aumenta. Com M = 8, por exemplo, obtém-se 100% de acerto.
  4. Regularização L2 (Tikhonov): Adicionar penalizador lambda na matriz de pesos de saída para suavizar o desenho da fronteira e prevenir overfitting.
  5. Treinamento Supervisionado Completo: Ajustar centros e spreads via gradiente descendente em conjunto com a camada de saída.
