### Task 1.2 — Create load_data.py

<read_first>
- adaline/training_data.csv
</read_first>

<action>
Create `adaline/load_data.py` with function `load_data(filepath)` that:
1. Reads the CSV with NumPy (skipping header)
2. Separates features X (columns x1–x4) and labels d (last column)
3. Prepends a bias column x0 = 1 to X → X_bias shape: (35, 5)
4. Returns `X_bias` (shape 35×5) and `d` (shape 35,)

```python
import numpy as np

def load_data(filepath='training_data.csv'):
    data = np.loadtxt(filepath, delimiter=',', skiprows=1)
    X = data[:, :4]          # x1, x2, x3, x4
    d = data[:, 4]            # labels: -1 or +1
    bias = np.ones((X.shape[0], 1))
    X_bias = np.hstack([bias, X])  # prepend x0=1 -> shape (35, 5)
    return X_bias, d

if __name__ == '__main__':
    X, d = load_data()
    print(f'X_bias shape: {X.shape}')   # (35, 5)
    print(f'd shape:      {d.shape}')    # (35,)
    print(f'Bias column (x0): {X[:3, 0]}')  # all 1.0
    print(f'First labels: {d[:5]}')
```
</action>

<acceptance_criteria>

- `d shape: (35,)`
- Bias column values all equal 1.0
</acceptance_criteria>
