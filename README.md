## 🚀 Projetos Desenvolvidos

### 1. Projeto Perceptron
Uma implementação teórica e prática do algoritmo de rede neural de camada única (Perceptron). O projeto aplica a Regra de Hebb Supervisionada para resolver um problema de classificação envolvendo amostras de óleo. Através da inicialização estocástica dos pesos, explora-se a convergência do modelo ao longo das épocas de treinamento, além de analisar as principais limitações das arquiteturas puramente lineares.

**📊 Acesse os Resultados (Dashboard do Perceptron):** [https://projeto-perceptron.vercel.app/perceptron/report.html](https://projeto-perceptron.vercel.app/perceptron/report.html)

> Você pode acessar as análises, as respostas às perguntas do trabalho e todas as tabelas de classificação e treinamento dentro da pasta `perceptron/` deste repositório.

### 2. Projeto ADALINE
Uma implementação teórica e prática da rede ADALINE (Adaptive Linear Neuron). O projeto aplica a regra de aprendizado de Widrow-Hoff (regra Delta) para classificar sinais ruidosos transmitidos a um comutador de válvulas industriais. O projeto explora a convergência do Erro Quadrático Médio (MSE) e a estabilidade dos pesos em problemas convexos.

**📊 Acesse os Resultados (Relatório ADALINE):** [https://projeto-perceptron.vercel.app/adaline/report.html](https://projeto-perceptron.vercel.app/adaline/report.html)

> Você pode acessar as análises de erro quadrático médio, resultados da classificação dos sinais e gráficos de convergência na pasta `adaline/` deste repositório.

### 3. Projeto RBF (rdl1)
Classificação de compostos radioativos (compostos químicos radioativos) usando redes neurais de Funções de Base Radial (RBF). Este projeto compara acurácias de aprendizado utilizando centros de RBF posicionados de forma aleatória vs. centros obtidos de maneira otimizada através do algoritmo K-Means.

**📊 Acesse os Resultados (Relatório RBF rdl1):** [https://projeto-perceptron.vercel.app/rdl1/index.html](https://projeto-perceptron.vercel.app/rdl1/index.html)

### 4. Projeto RBF2
Regressão não-linear para determinação da quantidade de gasolina a partir de parâmetros químicos do processo usando redes RBF. O projeto realiza um estudo de topologias (variando quantidade de neurônios ocultos), avalia o erro de treinamento e de generalização, e sugere a melhor arquitetura.

**📊 Acesse os Resultados (Relatório RBF2):** [https://projeto-perceptron.vercel.app/rbf2/report.html](https://projeto-perceptron.vercel.app/rbf2/report.html)

### 5. Projeto Hopfield (Memória Associativa)
Implementação de uma Rede de Hopfield com 45 neurônios para atuar como memória associativa bi-estável (auto-associativa). O sistema recupera 4 padrões de dígitos (1, 2, 3, 4) representados em grades de 9x5 bits após transmissão ruidosa por um canal de comunicação que corrompe cerca de 20% (ou mais) dos pixels de forma aleatória.

**📊 Acesse os Resultados (Relatório Hopfield):** [https://projeto-perceptron.vercel.app/hopfield/report.html](https://projeto-perceptron.vercel.app/hopfield/report.html)

> Você pode acessar o código da simulação, os testes unitários e o script de geração do dashboard na pasta `hopfield/` deste repositório.
