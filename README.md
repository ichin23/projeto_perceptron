## 🚀 Projetos Desenvolvidos

### 1. Projeto Perceptron
Uma implementação teórica e prática do algoritmo de rede neural de camada única (Perceptron). O projeto aplica a Regra de Hebb Supervisionada para resolver um problema de classificação envolvendo amostras de óleo. Através da inicialização estocástica dos pesos, explora-se a convergência do modelo ao longo das épocas de treinamento, além de analisar as principais limitações das arquiteturas puramente lineares.
